Some of my best apps and games

All those apps and games are made in Java with the JavaFX framework. The source code is available on github for free ;)

Logic Simulator

This is a simple circuit simulator made in Java with the JavaFX framework. Build whatever you want in this playground, even your own 8-bit pc

Rubik's Cube

Rubik's cube made with javafx3D for Windows, Linux, Mac and Android.
It can solve any 3x3 cube with the layers algorithm.

Reflection

A laser maze game.
Light up all the required targets by passing also through the checkpoints.

RailTheWay

Guide the trains to their station. Each train has a color that represents its destination. Be sure to avoid the cars on your way!

Retoohs

Retoohs is a reversed shooter game where you are the zombies and you have to kill the player and avoid letting them get points.

Food Dice

Food Dice is a food platformer where you are a dice and you get special effects. You have to reach the end without getting into lasers or shooters.

Chess

Chess GUI made in JavaFX for desktop, browser and android.
Singleplayer, LAN multiplayer and server multiplayer

Snake

Snake game (Bot included) using the Hamiltonian cycle and some pathfinding algorithms. You can play it alone, or wait till the perfect bot finishes the game

Clock

A simple clock made with JavaFX for Linux, Windows, Mac, Android and browser.
Click on the image to run it right now in your browser.
